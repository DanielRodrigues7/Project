Guia de Ajuda


Cadastro de Pacientes

Nesta aba do sistema é possivel cadastrar pacientes. O nutricionista somente poderá cadastrar uma dieta caso o paciente esteja cadastrado no sistema. Caso não haja pacientes cadastrados não será possivel visualiza-los na aba Cadastro de Dietas, nem nas demais abas, com isso, sugerimos que você começe por aqui.


Cadastro de Dietas

Nesta aba do sistema é possivel cadastrar dietas para os pacientes cadastrados na aba Cadastro de Pacientes. Você sera direcionado a uma tela onde é possivel cadastrar os grupos de alimentos assim como seus respectivos sub itens.


Diário Alimentar

Nesta aba é possivel visualizar todas as dietas que foram recomendadas para seus respectivos pacientes, assim como os dados de consumo que os pacientes cadastraram no sistema mobile, alem disso, tambem é possivel gerar um feedback que será visualizado via aplicativo pelo paciente.


Recomendações

Nesta aba do sistema é possivel cadastrar recomendações e visualizar as recomendações já cadastradas para seus pacientes, alem de gerar feedback de recomendações. Note que só estará disponivel para visualização e cadastro os pacientes ja cadastrados no sistema através da aba Cadastro de Pacientes.
